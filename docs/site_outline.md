# armaaxs Agency Site Outline

This document outlines the structure and content for the **armaaxs** agency website. The site should maintain a premium, high-tech aesthetic with interactive elements.

## Theme & Aesthetic

- **Core Branding**: "armaaxs" in `Explora` font.
- **Interactivity**: `InteractiveGridPattern` background with "flashlight" cursor effect.
- **Navigation**: Persistent `BottomDock` navigation bar.

---

## 1. Home (The Gateway)

- **Hero Section**: Massive "armaaxs" branding with the flashlight effect for immediate impact.
- **Micro-Services Row**: 3-4 key industry pillars (e.g., *Strategic Design*, *Scalable Architecture*, *Fluid UX*) appearing on scroll.
- **Featured Work (Teaser)**: High-fidelity thumbnails of top 2 projects in a horizontal scroll or subtle grid.
- **The "Why"**: A short, punchy paragraph explaining the philosophy: *"Crafting experiences, not just code."*
- **Call to Action**: A bold link or button leading to the Contact page.

## 2. Services (Specializations)

- **Custom Web Platforms**: Next.js, performance optimization, and global scalability.
- **Mobile Application Development**: iOS & Android development with a focus on fluid transitions and UX.
- **UI/UX Craftsmanship**: Motion design, premium aesthetics, and obsessive attention to user journeys.
- **Technical Strategy**: Consulting for startups and businesses on tech stack selection and architecture.

## 3. Work (Portfolio)

- **Project Grid**: Interactive thumbnail grid where projects "light up" on hover (utilizing the interactive grid logic).
- **Case Study Template**:
  - **The Challenge**: Defining the client's problem.
  - **The Solution**: How armaaxs engineered the solution.
  - **The Result**: Tangible outcomes (conversion rates, load times, user growth).
  - **Tech Stack**: Visual badges for technology used.
